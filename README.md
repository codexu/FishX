# FishX

头条摸鱼神器

## 使用方式

第一次使用，通过 ctrl+shift+p 打开命令面板，输入 FishX，选择 初始化 FishX。

## 功能

- 右下角状态栏切换新闻标题。
- 点击状态栏标题，可展示所有新闻列表。
- 点击左右按钮可以切换上一条或下一条新闻。
- 鼠标放置在 const 单词下即可展示内容。
- 鼠标放置在 export 单词下即可查看评论。
- 可扫码登录，精准摸鱼。
